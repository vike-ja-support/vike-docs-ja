新規でアプリケーションを立ち上げるには:

[Bati](https://batijs.dev/) を使用して、<Link href="/extensions">Vike 拡張機能</Link>を使用する新しい Vike アプリの雛形を作成することができます。

また、統合コードの完全な制御を望む場合は、以下のコマンドを使用して Vike 拡張機能を使用しない新しい Vike アプリの雛形を作成することもできます。

<Warning>
以下のコマンドは次の場合に限って推奨します：
 - 専門的なユーザーであり、統合コードの制御が必要な場合。
 - 好奇心が強く、統合コードを触ってみたい場合。
</Warning>

[npm](https://www.npmjs.com/)の場合:

```shell
npm init vike@latest
```

[pnpm](https://pnpm.io/)の場合:

```shell
pnpm create vike
```

[Yarn](https://yarnpkg.com/)の場合:

```shell
yarn create vike
```

オプション:

- `--skip-git`: 新しいGitリポジトリを初期化しない

ブロンプトで以下の選択肢から選択することができます：

- `react`: React + JavaScript
- `react-ts`: React + TypeScript
- `vue`: Vue + JavaScript
- `vue-ts`: Vue + TypeScript
