新規でアプリケーションを立ち上げるには:

[npm](https://www.npmjs.com/)の場合:

```shell
npm init vike@latest
```

[pnpm](https://pnpm.io/)の場合:

```shell
pnpm create vike
```

[Yarn](https://yarnpkg.com/)の場合:

```shell
yarn create vike
```

オプション:

- `--skip-git`: 新しいGitリポジトリを初期化しない

ブロンプトで以下の選択肢から選択することができます：

- `react`: React + JavaScript
- `react-ts`: React + TypeScript
- `vue`: Vue + JavaScript
- `vue-ts`: Vue + TypeScript
