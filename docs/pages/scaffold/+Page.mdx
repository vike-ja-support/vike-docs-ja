新規でアプリケーションを立ち上げるには:

You can use [Bati](https://batijs.dev/) to scaffold a new Vike app (that uses <Link href="/extensions">Vike extensions</Link>).

You can also, if you want full control over integration code, use the following command to scaffold a new Vike app (that doesn't use any Vike extension).

<Warning>
We recommend using the following command only if:
 - You are an expert user, and you need control over integration code.
 - You are curious and want to play with integration code.
</Warning>

[npm](https://www.npmjs.com/)の場合:

```shell
npm init vike@latest
```

[pnpm](https://pnpm.io/)の場合:

```shell
pnpm create vike
```

[Yarn](https://yarnpkg.com/)の場合:

```shell
yarn create vike
```

オプション:

- `--skip-git`: 新しいGitリポジトリを初期化しない

ブロンプトで以下の選択肢から選択することができます：

- `react`: React + JavaScript
- `react-ts`: React + TypeScript
- `vue`: Vue + JavaScript
- `vue-ts`: Vue + TypeScript
