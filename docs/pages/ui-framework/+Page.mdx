> The *UI framework* denotes the tool that renders your UI components. For example:
>  - [React](/react)
>  - [Vue](/vue)
>  - [Solid](/solid)
>  - [Angular](/angular)
>  - [Svelte](/svelte)
>  - [Preact](/preact)
>
> A UI framework essentially translates your declarative components's state changes into changes to the browser's DOM.
