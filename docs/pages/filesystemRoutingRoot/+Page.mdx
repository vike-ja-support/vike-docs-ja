The `filesystemRoutingRoot` setting allows you to change the URL root of <Link href="/filesystem-routing">Filesystem Routing</Link>. It's usually used for <Link href="/file-structure#domain-driven">domain-driven file structure</Link>.

<Warning>Don't use `filesystemRoutingRoot` to change the Base URL, see <Link href="/base-url" /> instead.</Warning>

```js
// /marketing/+config.js

export default {
  // Route to /* instead of /marketing/*
  filesystemRoutingRoot: '/'
}
```

```bash
PROJECT FILES                               URL
===============================             ======
/marketing/pages/index/+Page.js             /
/marketing/pages/about/+Page.js             /about
/marketing/pages/jobs/+Page.js              /jobs
```
